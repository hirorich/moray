# myGuiLib
***
## myGuiLib とは
- eelに代わるFlaskベースのGUI部品
- Flask, React, Bootstrap, Axios を使用する想定

***
## 環境構築
### Python
- python v.3.9
  - 環境作成
    ``` bash
    py -3.9 -m venv .venv
    ```
  - 以下のコマンドで必要なモジュールを一括インストール
    ``` bash
    pip install -r requirements.txt
    ```
