# moray
***
## moray とは
- eelに代わるFlaskベースのGUI部品
- 使用想定ライブラリ
  - Python
    - Flask 2.0
  - JavaScript
    - React 17
    - Axios
    - socket.io
  - CSS
    - Bootstrap 5

***
## 環境構築
### Python
- python v.3.9
  - 環境作成
    ``` bash
    py -3.9 -m venv .venv
    ```
  - 以下のコマンドで必要なモジュールを一括インストール
    ``` bash
    pip install -r requirements.txt
    ```
